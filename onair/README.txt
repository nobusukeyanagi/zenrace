AITuberデモ動画 統合HTMLパッケージ

内容
- index.html
- assets/ 配下のHTML・画像・音声・動画素材
- 削除済みの旧OBSシーンJSON（元のOBSシーンコレクション）

使い方
1. このZIPを解凍します。
2. OBSで「ブラウザ」ソースを追加します。
3. 「ローカルファイル」にチェックし、解凍したフォルダ内の index.html を指定します。
4. 幅 1080、高さ 2347 に設定します。
5. 音声もHTML内で再生されます。

タイムライン
- 0.000秒〜10.000秒：開始
- 10.000秒〜90.000秒：パーソナリティ2／ボイス_優勝戦開始
- 90.000秒〜160.000秒：コンシェルジュ／ボイス_優勝戦継続／チャット.htmlをこの画面開始時に再生開始
- 160.000秒〜175.427秒：パーソナリティ2に戻る
- 175.427秒〜246.933秒：パーソナリティ／ボイス_依存症対策
- 246.933秒〜256.933秒：開始に戻って終了

今回の反映内容
- シーン「開始」では、画面_時計のみを最上面で表示します。
- 背景_パーソナリティ.htmlは、青紫系の色合いを維持したレース場背景として同梱しています。
- ③コンシェルジュから④パーソナリティ2へ戻るタイミングを10秒早めています。
- ③コンシェルジュのチャット.htmlは、コンシェルジュ画面に入ったタイミングで読み直して再生開始します。
- アバターは、カラーキー #02f705／類似性700 相当の処理で背景透過します。
- ボイス再生中はアバター_トーク、ボイス停止中はアバター_ノーマルをループ再生します。

補足
- BGMは0秒から開始し、最後までループ再生します。
- BGM音量は index.html 内の CONFIG.bgmVolume で調整できます。初期値は 0.22 です。
- ブラウザの自動再生制限で音が出ない場合は、画面に出る START ボタンを押してください。
- アバターMP4はHTML側のcanvasでクロマキー処理しています。OBS側で別途カラーキーを設定する必要はありません。
